# Smart-delivery
## 🚚 Smart Delivery Optimization System

A full-stack web application inspired by modern logistics platforms like Amazon, designed to optimize delivery routes efficiently.

### 💡 Overview

This project simulates a real-world delivery system where multiple orders are processed and an optimized route is generated to minimize travel distance and improve delivery efficiency.

### 🔥 Key Features

* 📍 Add multiple delivery locations (latitude & longitude)
* ⚡ Priority-based routing (Prime vs Normal orders)
* 🧠 Route optimization using Greedy (Nearest Neighbor) algorithm
* 🗺️ Interactive map visualization using Leaflet
* 🔄 REST API integration between frontend and backend
* 📊 Estimated delivery distance calculation

### 🧠 Technical Highlights

* Implemented routing logic using algorithmic optimization techniques
* Designed clean backend architecture using Spring Boot
* Integrated frontend map UI with dynamic route rendering
* Demonstrated real-world system design concepts like delivery prioritization

### 🛠️ Tech Stack

* Backend: Java (Spring Boot)
* Frontend: React + Leaflet
* API: RESTful services
* Algorithm: Greedy (Nearest Neighbor)
